
## Sobre o Finance Api


## :rocket:Instalação
Para rodar o repositório é necessário clonar o mesmo, dar o seguinte comando para instalar as dependencias:

```bash
$ npm install
```

## Rodar o app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
